# pocket-terminal

Transforms your Raspberry Pi Zero into a fully portable, self-contained Linux terminal with a compact 1.44" LCD display and joystick controls. No external monitor, keyboard, or mouse required.

## Features

- Dual terminal sessions - switch between two independent terminals
- Background process execution with visual indicators
- On-screen virtual keyboard with full character support
- Command history navigation
- Hardware joystick controls
- Auto-start on boot
- One-command installation

## Installation

curl -sSL https://raw.githubusercontent.com/vassilis-ioannou2/pocket-terminal/main/install.sh | bash

## Hardware Required

- Raspberry Pi Zero / Pi Zero 2 (or any other pi)
- Waveshare 1.44" LCD HAT (https://www.waveshare.com/wiki/1.44inch_LCD_HAT)

## Controls

- Joystick: Navigate terminal/keyboard
- KEY1: Long press (5s) to shutdown
- KEY2: Switch between terminals
- KEY3: Execute command / Type character
